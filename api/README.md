# API example using proto files


## Setup

### Setup with pipenv
```
python3 -m venv ~/venv
source ~/venv/bin/activate
./setup.py develop
backyard-api
```

### Setup with pipenv
```
pipenv install --skip-lock
pipenv shell
./setup.py develop
backyard-api
```


## Usage

API explorer: http://localhost:8080/v1/ui/
Default credentials: admin/secret
